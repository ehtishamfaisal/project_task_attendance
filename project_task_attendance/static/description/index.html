Project Task Attendance

Select Project, Select Task, and add Description while checking in and checking out

This is an Attendance Apps. Update: v16.0.0.1

This module can help you to select a project on check-in. Also, it will prompt you to add a task for the selected project.
You can also add a description of the task, and in the future, it may also log your time on a specific task.

Select Project

Users should be able to choose a specific project while checking in.

Users should also be able to select a specific task related to the chosen project.

Enable users to add a description for their activities during both check-in and check-out.

Backend Modifications

Project, Task, Description are saved in attendance records of the respective user.

Data Validation

Users cannot check in and check out without selecting the fields.

Description Logs

Log history will be created upon adding a description.
